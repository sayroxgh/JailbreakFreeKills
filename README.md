# CS2 Jailbreak Freekill Plugin

A CounterStrikeSharp plugin for Counter-Strike 2 Jailbreak servers.

## Features
- Respawn dead Terrorist players
- Teleport players back to their death location
- Cooldown system for CT players
- Admin permission support
- Partial name targeting

## Commands
- `css_fk <player>` → Respawn a dead Terrorist player

## Requirements
- Metamod
- CounterStrikeSharp
- CS2 Dedicated Server
